Adding Labels with the Text Tool

The Text tool is how you add labels to the canvas. Text labels can be attached to pretty much anything. There are two ways to create text labels:
  • Click the Text tool, then click an object or a blank area of the canvas.
  • Double-click an object or a blank area of the canvas.
A new label starts out in text editing mode, so you can type or paste text into it. Click the Font button in the Style inspector to change the style of the selected text. Press Return when you’re done editing the label’s text. (To insert a line break instead, press Option-Return.) To go back and edit a label’s text later, just double-click it again.
