Selecting, Editing, and Moving Objects with the Modify Tool

The Modify tool works like the ordinary mouse pointer in most applications:
  • Click an object to select it. You can select individual points, or a whole line or fill including all of its points.
  • Hold the Command key while clicking an object to add it to or remove it from the selection. Holding Command also avoids automatically selecting all of the points in an object when selecting that object.
  • Hold the Shift key while clicking an object to add it to the selection. Holding Shift while clicking a point also selects all points on the same line between it and another selected point.
  • Drag an object to move it.
Once an object is selected, you can use the Style inspector to change its appearance.
