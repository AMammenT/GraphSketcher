Adding Labels with the Text Tool

Text labels are a great way to add further clarity to your visually realized data sets. They can be attached to points, lines, fills, axes, pretty much anything! If you’re in Draw or Fill mode, be sure to tap Done before continuing. Then, double-tap an object to attach a label to it; double-tap a blank area of the canvas to create a standalone label.
A new label starts out in text editing mode, so you can type text into it. To change the style of the label, use the Label Style popover. To go back and edit a label’s text later, just double-tap it.
