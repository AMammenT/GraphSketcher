Selecting, Moving, and Styling

If an object isn’t quite where you’d like it to be or its style isn’t all that stylin’, it’s easy to make a change. First, make sure you're not in Draw or Fill mode by tapping the Done button on the toolbar. (If it's not there, you're good to go!)
To move an object, touch and hold the object, then drag. To move an object one pixel at a time: touch and hold the object, then swipe with a second finger the direction in which you'd like the object to move.
To make style and property changes, tap the object you’d like to adjust. Tapping the object selects it; you can select individual points or a whole line or fill, including all of its points. To select all objects on the canvas: touch and hold the canvas; when its contextual menu appears, tap Select All. To select at least two (but not all) objects, drag across the canvas or touch and hold one object and tap the others. Once an object is selected, you can use the Info popover to change the selection’s appearance and attributes — for example, color — or to type in a precise location.
To delete an object: touch and hold the object; when its contextual menu appears, tap Delete.
To copy an object: touch and hold the object; when its contextual menu appears, tap Copy.
To paste an object: touch and hold the canvas; when its contextual menu appears, tap Paste.
You can also use the contextual menu to lock or unlock and detach objects. A locked object remains in its current position and cannot be moved or resized. An unlocked object can be moved and resized freely. Use detach to separate an object from an item to which it's attached so that it can be moved about without taking anything else along for the ride.
